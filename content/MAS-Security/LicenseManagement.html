授权服务

数据库

试用授权表（TrialLicense）

字段 数据类型 说明 约束
Id INT 授权记录标识符 主键，自增，唯一，不为Null
AppId VARCHAR(100) 应用程序唯一标识 不为Null，唯一
MachineCode VARCHAR(255) 机器码（绑定设备硬件） 不为Null
LicenseKey VARCHAR(255) 授权密钥 无
Type VARBINARY(64) 授权类型，加密存储（0 = 试用，1 = 正式） 不为Null
TrialDuration VARBINARY(64) 试用时长，加密存储（分钟） 不为Null
AccumulatedRunTime VARBINARY(64) 累计运行时长，加密存储（分钟） 不为Null
CreatedAt DATETIME 创建时间 不为Null
UpdatedAt DATETIME 最后更新时间 无
