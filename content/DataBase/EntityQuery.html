Entity Query 功能文档

概述

Entity Query是基于动态编译架构的数据库查询扩展模块。它允许用户通过导入预编译的实体DLL（包含数据库实体类），实现运行时解析实体元数据、生成查询SQL，并以分页形式在UI上展示数据。该模块无缝集成现有动态编译系统（构建模块），无需手动定义映射器——通过注解驱动的自动映射（反射提取[Table]、[Column]、[DisplayName]等），实现从DLL到DataGrid的端到端数据流动

功能

  • 从构建中编译的实体 DLL 导入
  • DLL 解析，枚举Assembly.GetTypes()，筛选带[Table]的类。提取表名、实体Type、属性元数据等
  • 数据库查询，通过动态仓储接口执行
  • 分页 UI 显示，DataGrid 绑定的动态 Type。启用AutoGenerateColumns=false，手动生成列
  • 表切换，ComboBox 显示解析出的表列表
  • 当前页数据导出、范围导出
  • 参数查询，模糊、精确查询
