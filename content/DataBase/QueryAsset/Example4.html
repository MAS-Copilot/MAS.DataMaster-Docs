按判定结果查询

本示例集中演示 enum（下拉框）参数的用法：候选项写在「说明」列，格式 选项: A,B,C，选中值替换进 SQL 时按字符串加单引号

使用说明

  • 本小节不含 SQL 代码块，解析时会被自动忽略
  • IsOk 字段 b'1' 为 OK、b'0' 为 NG；查询时间统一使用 CreatedAt
  • enum 的「默认值」需与某个候选项完全一致才能预选中。

按判定结果查询膜厚记录

按 OK / NG 判定筛选膜厚检测记录，演示最简单的二选一 enum

参数 标签 类型 默认值 说明
Result 判定结果 enum OK 选项: OK,NG 
SELECT
    f.ProductId   AS '产品ID',
    f.DetectionId AS '检测批次',
    ROUND(f.Thickness, 2) AS '膜厚(nm)',
    CASE f.IsOk WHEN b'1' THEN 'OK' ELSE 'NG' END AS '判定',
    f.CreatedAt   AS '检测时间'
FROM film_thickness_result f
WHERE CASE f.IsOk WHEN b'1' THEN 'OK' ELSE 'NG' END = {{Result}}
ORDER BY f.CreatedAt DESC;

按缺陷类型查询外观缺陷

enum 选项可多于两个，并可与其他类型参数（datetime、number）组合使用

参数 标签 类型 默认值 说明
DefectType 缺陷类型 enum 划痕 选项: 划痕,污点,崩边,气泡
StartTime 开始时间 datetime 2026-06-01 00:00:00 查询开始时间
EndTime 结束时间 datetime 2026-07-01 00:00:00 查询结束时间（不含）
Limit 查看记录数 number 100 返回前 N 条 
SELECT
    a.CreatedAt   AS '检测时间',
    a.ProductId   AS '产品ID',
    a.DetectionId AS '检测批次',
    a.DefectType  AS '缺陷类型',
    a.Severity    AS '严重等级',
    a.AreaMm2     AS '面积(mm²)'
FROM appearance_defect_point a
WHERE a.DefectType = {{DefectType}}
  AND a.CreatedAt >= {{StartTime}}
  AND a.CreatedAt <  {{EndTime}}
ORDER BY a.Severity DESC, a.CreatedAt DESC
LIMIT {{Limit}};
