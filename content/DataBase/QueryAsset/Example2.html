透过率检测

使用说明

  • 本小节不含 SQL 代码块，解析时会被自动忽略。
  • 查询时间统一使用 CreatedAt；IsOk 字段 b'1' 为 OK、b'0' 为 NG。

查询产品透过检历史

查看指定产品的所有透过率检测记录。

参数 标签 类型 默认值 说明
ProductId 产品 ID text PROD-001 要查询的产品编号 
SELECT
    d.ProductId   AS '产品ID',
    p.ProductName AS '产品名称',
    d.DetectionId AS '检测批次',
    d.RecipeNumber AS '配方编号',
    CASE d.IsOk WHEN b'1' THEN 'OK' WHEN b'0' THEN 'NG' ELSE '未知' END AS '最终判定',
    d.CreatedAt   AS '检测时间'
FROM transmittance_detection_result d
INNER JOIN Product_Info p ON d.ProductId = p.ProductId
WHERE d.ProductId = {{ProductId}}
ORDER BY d.CreatedAt DESC, d.Id DESC;

查询 NG 检测批次

查询指定时间段内最终判定为 NG 的透过率检测批次。

参数 标签 类型 默认值 说明
StartTime 开始时间 datetime 2026-06-01 00:00:00 查询开始时间
EndTime 结束时间 datetime 2026-07-01 00:00:00 查询结束时间（不含） 
SELECT
    d.CreatedAt   AS '检测时间',
    d.ProductId   AS '产品ID',
    d.DetectionId AS '检测批次',
    d.RecipeNumber AS '配方编号',
    'NG'          AS '最终判定'
FROM transmittance_detection_result d
WHERE d.IsOk = b'0'
  AND d.CreatedAt >= {{StartTime}}
  AND d.CreatedAt <  {{EndTime}}
ORDER BY d.CreatedAt DESC, d.Id DESC;

查询透过率最低的点位

查询指定时间段内透过率最低的点位，适合排查异常点位。

参数 标签 类型 默认值 说明
StartTime 开始时间 datetime 2026-06-01 00:00:00 查询开始时间
EndTime 结束时间 datetime 2026-07-01 00:00:00 查询结束时间（不含）
Limit 查看记录数 number 100 返回前 N 条最低点位 
SELECT
    r.CreatedAt   AS '检测时间',
    r.ProductId   AS '产品ID',
    r.DetectionId AS '检测批次',
    r.PointIndex  AS '点位序号',
    ROUND(r.Transmittance, 3) AS '透过率'
FROM transmittance_point_result r
WHERE r.CreatedAt >= {{StartTime}}
  AND r.CreatedAt <  {{EndTime}}
ORDER BY r.Transmittance ASC
LIMIT {{Limit}};
