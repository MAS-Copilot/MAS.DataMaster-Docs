膜厚检测

查询产品膜厚检测历史

查看指定产品的所有膜厚检测记录；产品 ID 可空，留空时查询全部产品。

参数 标签 类型 默认值 是否可空 说明
ProductId 产品 ID text PROD-001 true 要查询的产品编号，留空查询全部 
SELECT
    f.ProductId   AS '产品ID',
    f.DetectionId AS '检测批次',
    ROUND(f.Thickness, 2) AS '膜厚(nm)',
    CASE f.IsOk WHEN b'1' THEN 'OK' WHEN b'0' THEN 'NG' ELSE '未知' END AS '判定',
    f.CreatedAt   AS '检测时间'
FROM film_thickness_result f
WHERE ({{ProductId}} = '' OR f.ProductId = {{ProductId}})
ORDER BY f.CreatedAt DESC;

按日期统计膜厚趋势

查看每天膜厚的平均值、最大值和最小值变化。

参数 标签 类型 默认值 说明
StartTime 开始时间 datetime 2026-06-01 00:00:00 查询开始时间
EndTime 结束时间 datetime 2026-07-01 00:00:00 查询结束时间（不含） 
SELECT
    DATE(f.CreatedAt) AS '日期',
    COUNT(*)          AS '检测次数',
    ROUND(AVG(f.Thickness), 2) AS '平均膜厚',
    ROUND(MIN(f.Thickness), 2) AS '最小膜厚',
    ROUND(MAX(f.Thickness), 2) AS '最大膜厚'
FROM film_thickness_result f
WHERE f.CreatedAt >= {{StartTime}}
  AND f.CreatedAt <  {{EndTime}}
GROUP BY DATE(f.CreatedAt)
ORDER BY DATE(f.CreatedAt) ASC;
