查询缺陷点位明细

查看某个产品某次检测批次下每个缺陷的位置、类型与严重等级。

参数 标签 类型 默认值 说明
ProductId 产品 ID text PROD-001 要查询的产品编号
DetectionId 检测批次 ID text DETECT-001 要查询的检测批次 
SELECT
    a.DefectIndex AS '缺陷序号',
    a.DefectType  AS '缺陷类型',
    a.X           AS 'X坐标',
    a.Y           AS 'Y坐标',
    a.Severity    AS '严重等级',
    a.AreaMm2     AS '面积(mm²)'
FROM appearance_defect_point a
WHERE a.ProductId   = {{ProductId}}
  AND a.DetectionId = {{DetectionId}}
ORDER BY a.Severity DESC, a.DefectIndex ASC;

查询缺陷类型分布

统计指定时间段内各类外观缺陷的出现次数与占比。

参数 标签 类型 默认值 说明
StartTime 开始时间 datetime 2026-06-01 00:00:00 查询开始时间
EndTime 结束时间 datetime 2026-07-01 00:00:00 查询结束时间（不含） 
SELECT
    a.DefectType AS '缺陷类型',
    COUNT(*)     AS '出现次数',
    CONCAT(ROUND(COUNT(*) * 100.0 /
        (SELECT COUNT(*) FROM appearance_defect_point), 2), '%') AS '占比'
FROM appearance_defect_point a
WHERE a.CreatedAt >= {{StartTime}}
  AND a.CreatedAt <  {{EndTime}}
GROUP BY a.DefectType
ORDER BY 出现次数 DESC;
