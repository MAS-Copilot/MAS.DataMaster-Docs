后端（C#）调用前端（JavaScript）的函数

后端通过 IMASCodeService 接口与前端进行交互，主要通过执行 JavaScript 脚本的方式实现。以下是后端调用前端的方法列表：

C# 方法名 前端函数名 描述 调用时机
DisplayUserMessageAsync(string content) addUserMessage(content) 在聊天窗口中追加一条用户消息气泡 用户发送消息后
StartAssistantStreamAsync(string streamId) startAssistantStream(streamId) 创建一条新的助手流式消息容器并显示加载指示 流式响应开始前
UpdateAssistantStreamAsync(string streamId, string chunk) updateAssistantStream(streamId, chunk) 向指定流式消息追加内容片段，并触发 Markdown 渲染与代码高亮 流式响应过程中（每个 chunk）
EndAssistantStreamAsync(string streamId) endAssistantStream(streamId) 结束流式消息，移除 loading 状态；内容为空时显示“（已取消）” 流式响应结束 / 中断
ClearConversationAsync() clearAllMessages() 清空当前对话消息列表，并清理流式消息缓存 用户新建会话 / 清空会话
ShowToastMessageAsync(string content, InfoLevel level) showToastMessage(content, level) 显示 Toast 提示（success / info / warning / error） 网络异常、状态提示等
CaptureLongScreenshotAsync() captureLongScreenshot() 对当前会话生成长截图并下载 用户点击“长截图”按钮
SetThemeAsync(AppTheme theme) setTheme(theme) 设置前端主题（仅支持 dark / light） 应用启动、主题切换、跟随系统主题变化时
SetBlurActiveAsync(bool isActive) setBlurActive(isActive) 设置前端模糊遮罩层 WindowBlurHelper.BlurStateChanging 触发时
