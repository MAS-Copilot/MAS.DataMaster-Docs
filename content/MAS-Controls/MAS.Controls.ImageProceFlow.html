边框颜色定义

  • 灰色（Gray）：默认
  • 绿色（Green）：初始化成功或正常状态
  • 红色（Red）：执行过程中发生了异常
  • 橙色（Orange）：输入数据存在问题
  • 紫色（Purple）：初始化失败
  • 黄色（Yellow）：警告或非致命错误
private Brush _borderBrush = Brushes.Gray;
/// <summary>
/// 获取或设置控件边框颜色
/// </summary>
public Brush BorderBrush {
    get => _borderBrush;
    set => SetField(ref _borderBrush, value, nameof(BorderBrush));
}
