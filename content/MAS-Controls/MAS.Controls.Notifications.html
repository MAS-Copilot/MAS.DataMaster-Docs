数据库

通知记录（NotificationRecord）

字段 数据类型 说明 约束
Id INT 通知记录的标识符 主键，自增，唯一，不为Null
Title VARCHAR(100) 通知标题 不为Null
Message TEXT 通知内容 不为Null
Type VARCHAR(20) 通知类型 不为Null
IsRead BIT 通知是否已读（0-未读，1-已读） 不为Null，默认0
ResourceId VARCHAR(100) 通知关联的资源ID 无
CreatedAt DATETIME 创建时间 不为Null
UpdatedAt DATETIME 最后更新时间 无
