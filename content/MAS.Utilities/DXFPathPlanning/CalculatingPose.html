姿态生成系统设计

本模块将离散的几何路径点（XYZ）转化为包含完整六自由度信息的机器路径（XYZ + RxRyRz）。为了计算出物理意义正确的旋转角度，算法强依赖于以下前置数据的完整性与准确性

计算流程概览

  • 加载 DXF 文件（参见 DXFParser）
  • 按图层组织实体集合数据
  • 移除属于忽略组的几何实体
  • 限制生成过多轨迹点
  • 将几何实体转换为轨迹点
  • 执行路径优化，生成最优轨迹序列（参见 OptimizationPath）
  • 从最终轨迹生成姿态(在本文档解释范围)
  • 确保相对位置
  • 得到该图层完整的轨迹数据

姿态规则前置

Tip

关于正交坐标系

\(T, B, N_{eq}\) 这三个向量构成一个局部工具坐标系

  • \(T\)（Tangent → 切向）: 工具的相对前后

  • \(B\)（Binormal → 副法向）: 工具的相对左右

  • \(N_eq\)（Equivalent Normal → 等效法向）: 工具的相对上下

只要这三个向量互相垂直且都归一化，就得到一个完美无翻转的工具姿态

定义机器人在路径上的行为策略及数学解算标准。此部分数据通常来源于TrajectoryParameters配置

旋转顺序

机器人的欧拉角解算强依赖于旋转轴的叠加顺序。不同的控制器标准不同

  • 标准：通用顺序多为ZYX（Yaw-Pitch-Roll）或XYZ（Roll-Pitch-Yaw）
  • 输入：必须在配置文件中指定目标控制器的旋转顺序
  • 影响：若顺序错误，计算出的 \((Rx, Ry, Rz)\) 会导致机器做出完全不可预测的姿态
Warning

注意事项

当前姿态求解在内部统一使用 ZYX 约定来还原欧拉角

生成策略

决定工具姿态如何随路径变化

  • Fixed (固定姿态)：适用于点胶、三轴雕刻。全过程保持工具垂直或特定角度，忽略路径方向
  • Tangent Following (切向跟随)：适用于切割、刀片裁剪、焊接。\(Rz\) (Yaw) 始终指向路径行进方向
  • Surface Normal (法向跟随)：适用于需要姿态随空间路径实时变化的工艺（打磨、3D 曲线加工等）

若没有真实曲面，就无法生成贴合物理曲面的 Rx/Ry。此时系统采用 GlobalUp 等工艺约定，构造等效法向并自动计算 Rx/Ry，使姿态满足工艺需求

Note

常用法向构造解法（无真实曲面时）

已知：\(T\)（路径切线），\(U\) = GlobalUp（工艺参考 Up） 步骤：

  1. 去掉 \(U\) 在 \(T\) 方向上的分量：\(U_{proj} = (U\cdot T)T\)
  2. 得到 \(U\) 在法平面上的投影：\(N_{raw} = U - U_{proj}\)
  3. 归一化 \(N_{raw}\)，得到等效法向 \(N_{eq}\)
  4. 再用 \(B = T \times N_{eq}\) 得到第三个轴，构成正交坐标系 \(T, B, N_{eq}\)

零位定义

定义“什么都不转”时的物理状态

  • 垂直向下参考值：例如，对于许多 6 轴机器人，\((Rx=180, Ry=0, Rz=0)\) 代表法兰盘垂直指向地面
  • 数据需求：需要配置BaseRx, BaseRy, BaseRz作为计算的基底值

几何数据前置

定义从 DXF 图元中提取的数学向量。此部分数据来源于DxfLayerEntities解析结果

切向向量

  • 定义：当前点指向下一个点的单位向量
  • 计算公式：\(\vec{T} = \text{Normalize}(P_{next} - P_{current})\)
  • 用途：决定 \(Rz\) (偏航角/Yaw)
  • 约束：点集必须是有序的（Ordered）。对于圆弧，离散密度必须足够高以保证向量变化平滑

法向量

  • 定义：几何体所在平面的垂直向量
  • 来源：
    • 2D DXF：默认为 \(\vec{N} = (0, 0, 1)\)
    • 3D Surface：需从 CAD 模型表面投影获取 \((Nx, Ny, Nz)\)
  • 用途：决定 \(Rx\) (滚转角/Roll) 和 \(Ry\) (俯仰角/Pitch)
  • 约束：若法向量错误，机器人可能尝试从侧面切入平面

路径平滑度

  • 要求：路径中是否存在尖锐拐角（Sharp Corner）
  • 处理：在尖角处，切向向量会发生突变（如 90 度）。需要做圆角过渡（Fillet）或单独旋转点

补偿与校准前置

定义物理硬件与理论模型之间的偏差。此部分数据来源于现场校准参数

工具安装角度

工具本身安装在法兰盘上时的旋转偏差

  • 场景：切刀可能安装在法兰盘侧面，或者刀刃默认朝向与法兰盘 X 轴呈 90 度
  • 输入：
    • ToolMountingRx (deg)：工具相对于法兰盘工具坐标系绕 X 轴的固定旋转偏差（度）
    • ToolMountingRy (deg)：工具相对于法兰盘工具坐标系绕 Y 轴的固定旋转偏差（度）
    • ToolMountingRz (deg)：工具相对于法兰盘工具坐标系绕 Z 轴的固定旋转偏差（度，通常用于补偿刀刃方向与法兰盘默认 X 轴的夹角）
  • 算法动作：在计算出的理论角度上叠加此偏移量

工艺角

工艺需要的额外倾斜角

  • 场景：焊接枪不能垂直焊接，需要前倾 15 度（Push angle）
  • 输入：LeadAngle (进给倾角), TiltAngle (侧倾角)
  • 算法动作：基于切向坐标系（Frenet Frame）进行局部旋转叠加
