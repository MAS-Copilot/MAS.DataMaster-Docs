几何实体解析

正则表达式

正则表达式用于匹配每个图层中，使用文本注解的特定扩展点

注意事项

  • 文本类型分为Text（单行）和MText（多行）类型的文本
  • 注解解析对大小写不敏感

注解

标题 文本 描述 是否必须 实体类型
原点 Origin: X = value, Y = value, Z = value 设置该图层的原点坐标，排除在轨迹点外 否 单行文本
扩展点 Name-index: X, Y, Z 设置该图层的扩展点坐标，排除在轨迹点外 否 多行文本 or 单行文本

原点

  • 原点坐标通过解析图层中的单行文本提取
  • 每个图层有且只有一个，如果解析存在多个，则返回列表的第一个
  • 示例：3D: Origin: X = 10.0, Y = 20.0, Z = 0.0, 2D: Origin: X = 10.0, Y = 20.0

扩展点

  • 扩展点坐标通过解析图层中的多行文本提取
  • 扩展点映射列表：
    • 拍照点 => PhotoPoint
    • 抓取点 => GrabPoint
    • 放置点 => PlacementPoint
    • 结合点 => CombinePoint
    • 作业点 => OperationPoint
  • 每个图层可以存在多个，如果单个图层的Name-Index存在重复，只使用重复列表的第一个
  • 示例（顺序排序 X,Y,Z）：
    • 拍照点-1: 10.0, 20.0, 0.0
    • PhotoPoint-2: 10.0, 20.0, 0.0

组

几何忽略

  • 在 CAD 软件中，将不需要解析的线段分配到任意的组
  • 组（Group）用于组织图层中的几何体，以控制解析过程，决定哪些几何体需要被忽略
  • 几何体可以分配到任意组（没有命名要求），以标记为不需要解析
  • 选择忽略的几何 -> 右键 -> 组
