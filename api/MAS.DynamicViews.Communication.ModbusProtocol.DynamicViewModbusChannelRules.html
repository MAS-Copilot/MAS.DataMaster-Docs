CanWrite(string)

判断指定地址区域是否允许写入操作

public static bool CanWrite(string addressArea)

Parameters

addressArea string

要检查的地址区域（例如 DynamicViewModbusAddressAreas.COILS）

Returns

bool

如果该地址区域支持写操作（线圈和保持寄存器），返回 true；否则返回 false

Remarks

根据 Modbus 协议标准：

  • COILS（功能码 05、15）可读写
  • HOLDING_REGISTERS（功能码 06、16）可读写
  • 离散输入和输入寄存器为只读
