StabilityFrames

获取或设置姿态稳定滤波所使用的窗口帧数

对连续帧的旋转姿态（Quaternion）进行窗口均值滤波， 在 SO(3) 空间内抑制抖动与瞬时跳变。

取值说明： 0 或 1：不启用稳定滤波，使用单帧即时结果； 2~4：低延迟稳定（适合低帧率场景）； ≥ 5：强稳定，但会引入明显时间延迟

默认值：1
