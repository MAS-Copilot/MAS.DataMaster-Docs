动态视图变量值提供者

public interface IDynamicViewVariableValueProvider

Remarks

只负责提供运行时变量值与结构定义

主要用于：

  • 运行时绑定
  • UI显示
  • 状态监控
  • 表达式计算
  • 运行时事件读取
