Remarks

通用 JSON 节点操作，不含 ChartAsset 业务语义。支持的 Op：

  • replace：替换 Path 指向的已存在节点，值取自 ValueJson
  • remove：删除 Path 指向的已存在节点
  • append：Path 必须指向数组，把 ValueJson 追加到末尾
  • insertAfter：Path 必须指向数组中的现有元素，在其后插入 ValueJson
Path 是 camelCase 资产 JSON 上的 JSON Pointer，数组节点可用下标或通用 byId/{id} 段（按元素的 id 字段定位）导航，例如 /dashboard/widgets/byId/trend/option/series/0/type
