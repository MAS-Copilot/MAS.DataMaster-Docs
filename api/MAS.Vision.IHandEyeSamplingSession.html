Parameters

robotPoses List<Pose6D>

机器人位姿序列（Base → Gripper）

visionPoses List<Pose6D>

视觉位姿序列（Camera → Target）

robotRotationOrder RotationOrder

机器人欧拉角旋转顺序

cameraRotationOrder RotationOrder

相机欧拉角旋转顺序

Returns

int

当前已累计的样本总数量

Remarks

两个列表必须满足以下条件：

  • 长度相同
  • 索引一一对应
  • 表示同一时刻采集到的机器人位姿与视觉位姿

位姿语义约定：

  • robotPoses：Base → Gripper（或 Tool）的位姿
  • visionPoses：Camera → Target（标定板/靶标）的位姿
