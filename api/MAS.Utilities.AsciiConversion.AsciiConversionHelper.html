ConvertFromHex(string)

将十六进制编码字符串转换为ASCII文本

public static string ConvertFromHex(string value)

Parameters

value string

十六进制输入字符串，支持格式：

  • 单值：如 "41" 或 "0x41"（对应 'A'）
  • 多值：使用空格或逗号分隔，如 "41 42" 或 "0x41,0x42,43"（对应 "ABC"）
  • 大小写不敏感：支持 "aF" 或 "AF"（均对应 175）

Returns

string

转换后的ASCII文本

Exceptions

FormatException

输入包含无效十六进制值（如非0-9、A-F字符或无效格式）
