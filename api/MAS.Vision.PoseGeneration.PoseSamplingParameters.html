AngleToPositionWeightMmPer

获取或设置角度差折算为等效位置差的权重（单位：mm/deg）

[ExcludeFromCodeCoverage]
public double AngleToPositionWeightMmPer { get; set; }

Property Value

double

Remarks

在综合评估位姿间“空间距离 + 姿态差异”时， 用于将角度差（deg）转换为等效的位置距离（mm）， 以便在统一度量空间中进行比较

数值越大，姿态差异在综合距离中的权重越高

默认值：5.0

CandidatePoolSize

获取或设置随机候选池大小

[ExcludeFromCodeCoverage]
public int CandidatePoolSize { get; set; }

Property Value

int

Remarks

表示在每一轮采样中随机生成的候选位姿数量。 候选池越大，越有利于获得分布均匀的结果，但计算开销也会相应增加

默认值：300。推荐范围：200 ~ 800
