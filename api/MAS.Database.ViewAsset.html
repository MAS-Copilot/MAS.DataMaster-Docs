ViewAssetInfo

ViewAssetManifest

视图资产 manifest.json 模型。只描述资产，不限制前端实现

ViewAssetManifestReader

视图资产 manifest.json 读取与解析

ViewAssetPaths

视图资产路径解析与安全校验

ViewAssetPreviewResult

视图资产预览结果（供 Agent 使用）。额外提供 Records，默认截断为 50 行

ViewAssetProjectModel

视图资产工程持久化模型

ViewAssetQueryException

视图资产查询在数据库执行阶段失败时抛出。 消息包含 SQL 文件名与脱敏后的底层错误（password/pwd 键值会被替换为 ***）， 可直接返回给前端；区别于请求或参数不合法时的 InvalidOperationException

ViewAssetQueryRequest

视图资产查询请求。为安全起见只允许指定资产目录内的 .sql 文件，禁止直接传 SQL 文本

ViewAssetQueryRequestParser

视图资产查询请求解析：HTTP 页面查询与 IPC viewAsset.preview 共用的 parameters 解析与校验实现，保证两条入口行为一致。 参数值保留 JSON 原生类型（字符串/数字/布尔/null）， 也支持 {"type":"datetime","value":"2026-07-01T08:30:00+08:00"} 显式类型包装

ViewAssetQueryResult

视图资产查询结果，以 columns + rows 形式返回前端

ViewAssetRefreshPolicy

视图资产刷新策略

ViewAssetSqlGuard

视图资产只读 SQL 守卫，规则与 QueryAsset / ChartAsset 保持一致：只允许单条只读语句

ViewAssetUpsertRequest

视图资产注册请求。资产文件先由 CLI 创建到资产目录，再通过本请求登记到工程索引

ViewAssetValidationResult

视图资产校验结果

ViewAssetValidator

视图资产校验：目录结构、入口文档、清单与 SQL 只读性
