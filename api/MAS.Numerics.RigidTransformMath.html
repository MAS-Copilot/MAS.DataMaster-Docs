InvertRigid(double[,], double[,])

计算刚体变换（R, t）的逆变换

对于刚体变换： 

X' = R * X + t
其逆变换为： 
X = R^T * (X' - t)

因此：

  • R⁻¹ = Rᵀ
  • t⁻¹ = -Rᵀ · t
public static (double[,] RInv, double[,] TInv, double[,] HInv) InvertRigid(double[,] r3x3, double[,] t3x1)
