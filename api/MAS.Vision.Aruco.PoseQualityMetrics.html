DetectedMarkerCount

获取成功检测到的 ArUco Marker 数量

Marker 数量过少通常意味着可用于姿态解算的几何约束不足， 可能导致姿态结果不稳定或不可信。

可结合最小 Marker 数阈值，用于判定该帧是否有效。

ReprojectionErrorPx

获取姿态解算后的重投影误差（单位：像素）

表示将三维点通过解算得到的位姿重新投影回图像后， 与实际检测到的图像点之间的平均误差。

该值越小，说明解算结果与观测数据越一致， 姿态结果的可信度越高。

当该值为 null 时，表示当前解算流程中 未计算或未启用重投影误差评估。 

public double? ReprojectionErrorPx { get; init; }
