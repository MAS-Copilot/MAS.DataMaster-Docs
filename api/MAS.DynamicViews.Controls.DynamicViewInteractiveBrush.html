Remarks

设置 Background/BorderBrush 后自动按亮度派生悬停（约 8%）与按下（约 15%）状态画刷： 浅色变暗、深色变亮。配合 DynamicViewInteractiveButton.xaml 模板使用， 状态色由模板 Trigger 作用于模板内部 Border，不与控件本地 Background 争夺属性优先级。 来源画刷通过 SetResourceReference 关联资源 Key 时，主题切换会触发派生色自动重算
