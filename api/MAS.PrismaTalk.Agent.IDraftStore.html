Interface IDraftStore

Namespace
MAS.PrismaTalk.Agent
Assembly
MAS.PrismaTalk.dll

通用临时草稿存储，面向"强校验 + 大文本"场景。 模型先把完整内容存为草稿，之后用 draftId 引用并做小步编辑，避免每轮重传大文本

public interface IDraftStore

Remarks

草稿只落盘到本机临时目录；靠 TTL 与最大数量兜底清理 存储目录对同一用户的所有进程一致，因此 PrismaTalk（agent 进程）写入的草稿 可被 DataMaster（宿主进程）内的校验/入库工具直接读取
