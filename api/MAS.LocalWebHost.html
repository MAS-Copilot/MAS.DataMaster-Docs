Classes

LocalWebAccessToken

本地 Web Host 访问 token。时间均为 UTC

LocalWebHostHandle

本地 Web Host 启动后的句柄，描述实际监听地址、访问模式与访问 token

LocalWebHostOptions

本地 Web Host 启动配置

LocalWebHostStateChangedEventArgs

本地 Web Host 生命周期状态变更事件参数

LocalWebRequestContext

本地 Web Host 请求上下文。对 ASP.NET Core HttpContext 的封装， 使模块无需感知底层 HTTP 类型即可读取路由、请求体并写入响应

ServiceCollectionExtensions

提供扩展方法，用于在 IServiceCollection 中注册本地 Web Host 服务

Interfaces

ILocalWebAccessTokenService

本地 Web Host 访问 token 服务。Host 级 token：一个有效 token 可访问当前 Host 下全部模块

ILocalWebEndpointModule

允许模块直接向 ASP.NET Core 管道注册 endpoints/middleware。 注册的管道在通用路由分发（/modules、/api）之前执行， 且不经过宿主的 token 校验，模块需自行处理访问控制

ILocalWebHostService

通用本地/局域网 Web Host 服务。基于 Kestrel，负责静态文件与模块请求分发， Lan 模式可按启动配置启用访问 token 校验

ILocalWebModule

本地 Web Host 模块。宿主只负责路由分发与通用能力，具体资源解析由模块实现

LocalWebHostAccessMode
