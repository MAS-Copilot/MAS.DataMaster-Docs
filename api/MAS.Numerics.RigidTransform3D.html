构建完成的 RigidTransform3D 实例

Remarks

该方法假定 r3x3 为正交旋转矩阵（R·Rᵀ≈I，det(R)≈1）。 平移向量采用列向量形式：T = [x y z]ᵀ， 齐次矩阵 H 的结构为： 

[ R00 R01 R02 x ]
[ R10 R11 R12 y ]
[ R20 R21 R22 z ]
[  0   0   0  1 ]

同时会计算并填充逆变换（Inverse）， 其中 R⁻¹ = Rᵀ，T⁻¹ = −Rᵀ·T。

ValidateOrThrow(string?)

校验当前刚体变换的数据有效性与基本数学性质

public void ValidateOrThrow(string? name = null)

Parameters

name string

可选名称前缀，用于异常信息定位（如 "BaseToGripper"）

Remarks

确保刚体变换满足以下约束：

  • R 为 3×3 矩阵
  • T 为 3×1 列向量
  • H 为 4×4 齐次矩阵
  • R 近似正交：‖R·Rᵀ − I‖₂ ≤ 1e−2
  • det(R) ≈ 1（纯旋转，无缩放/镜像）

ArgumentException
