ExpandByEpsilon(Rect, double)

按指定容差值对称扩展矩形边界

public static Rect ExpandByEpsilon(this Rect rect, double epsilon)

Parameters

rect Rect

需要扩展的原始 Rect

epsilon double

容差值（单位：毫米），扩展量将以原矩形中心为对称轴向外平均分配

Returns

Rect

返回扩展后的 Rect，其中心与原矩形一致，宽度和高度各增加 2 * epsilon

ExpandByPercentage(Rect, double)

按指定百分比对称扩展矩形边界

public static Rect ExpandByPercentage(this Rect rect, double percentage)

Parameters

rect Rect

需要扩展的原始 Rect

percentage double

扩展百分比，典型取值如 0.02 表示在宽度和高度方向上分别扩大 2%， 扩展量将以原矩形中心为对称轴向外平均分配（即四周均匀扩展）

Returns

Rect

返回扩展后的 Rect，其中心与原矩形一致，宽度和高度均增加了指定百分比
