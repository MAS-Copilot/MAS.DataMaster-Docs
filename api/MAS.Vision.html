AxisRef

OpenCV 相机坐标系下的轴方向引用

用于描述“实际/目标坐标系的某一轴” 对应 OpenCV 相机坐标系中的哪一根轴及其方向

OpenCV 相机坐标系约定（右手系）： - X：图像水平方向向右 - Y：图像垂直方向向下 - Z：相机光轴方向，指向场景前方

例如： - 若实际 X 轴定义为“相机朝前”，应选择 CvZPos - 若实际 Z 轴定义为“图像向上”，应选择 CvYNeg
