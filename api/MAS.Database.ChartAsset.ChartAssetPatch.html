通用的结构化 JSON 补丁工具，基于 Newtonsoft.Json.Linq.JToken 对 JSON 树做局部修改。 只理解 JSON 路径与节点操作，不包含 ChartAsset 业务语义（校验仍由 ChartAssetValidator 承担）

Remarks

支持的操作：

  • replace：替换 path 指向的已存在节点；路径不存在即失败，不会静默创建字段
  • remove：删除 path 指向的已存在节点
  • append：path 必须指向数组，把值追加到数组末尾
  • insertAfter：path 必须指向数组中的现有元素，在其后插入值

路径是标准 JSON Pointer，并额外支持通用 byId/{id} 段：当前节点为数组时， 按元素的 id 字段定位（适用于任何元素含 id 的数组），也支持数组下标
