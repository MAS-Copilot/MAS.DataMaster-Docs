GetOffsetMatrix(Point, double, Point)

获取包含平移和旋转的 3x3 偏移矩阵

public static Matrix<double> GetOffsetMatrix(Point offset, double rotationAngle, Point rotationCenter)

Parameters

offset Point

平移偏移量 (dx, dy)，单位：与坐标一致

rotationAngle double

绕 Z 轴旋转角度（度），遵循右手坐标系，逆时针为正

rotationCenter Point

旋转中心点

Returns

Matrix<double>

3x3 偏移矩阵 T_offset

Remarks

矩阵形式： [ cosθ -sinθ dx + cx - cxcosθ + cysinθ ] [ sinθ cosθ dy + cy - cxsinθ - cycosθ ] [ 0 0 1 ] 其中 (cx, cy) 是旋转中心
