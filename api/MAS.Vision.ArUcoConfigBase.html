MinMarkers

获取或设置最小 Marker 数量门限

[ExcludeFromCodeCoverage]
public int MinMarkers { get; set; }

Property Value

int

Remarks

当检测到的 marker 数小于该值时，认为该帧质量不足，姿态解算应判定失败或不输出

该门限用于避免误检导致的“看似有角度但实际错误”的结果

默认值：1
