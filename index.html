DataMaster

🚀 项目简介

DataMaster 是 MAS 自动化体系中的核心上位机平台，用于工业设备的数据管理、视觉处理、流程编排与设备通信。系统基于 .NET / C# 与 WPF 构建，采用模块化架构设计，支持视觉算法、路径规划、设备通信以及数据持久化等多种工业应用场景

🛠️ 开发者模式

开发者模式用于调试与扩展 DataMaster 系统功能，在该模式下将启用部分开发辅助能力，包括：

  • DM 视觉工作台中的源文件编辑
  • 数据库中定义数据交换的引用程序集、自定义属性显示
  • Prisma 配置中使用环境变量为MAS.PrismaTalk_API的明文密钥
  • 配置页面中显示每个单独的修改次数
  • DXF 解析中的源文件编辑
  • 数据库使用环境变量为DB_PASSWORD的明文密码
  • Biud 中的源文件编辑
  • Word 打印中的源文件编辑
  • VM 视觉工作台中受保护项目的源文件保留
